# Microsoft Foundry Introduction

## Summary
Microsoft Foundry is presented as an end-to-end AI development ecosystem that supports model deployment, agents, evaluation, observability, governance, and enterprise AI workloads.

## Key Concepts
- Microsoft Foundry
- Model Catalog
- Agent Service
- Foundry IQ
- Observability
- Security and Governance
- Serverless Inferencing

## Definitions
### Model Catalog
Repository containing thousands of deployable AI models from multiple vendors.

### Foundry IQ
Knowledge consolidation layer used to power AI agents.

### Agent Service
Platform capability for building tool-enabled AI agents.

## Examples
- Deploy OpenAI, Meta, Anthropic, and Grok models.
- Build agents over SharePoint and M365 data.

## Step-by-Step Process
1. Deploy model from catalog.
2. Create agent.
3. Connect knowledge sources.
4. Add tools.
5. Evaluate and monitor.
6. Secure and govern workloads.
