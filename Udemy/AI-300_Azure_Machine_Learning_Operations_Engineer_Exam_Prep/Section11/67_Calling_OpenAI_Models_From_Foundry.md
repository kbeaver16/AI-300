# Calling OpenAI Models from Foundry SDK

## Summary
Lab demonstrating use of Azure AI Projects SDK, Azure Identity, and OpenAI SDK to call GPT deployments hosted in Foundry.

## Key Concepts
- Azure AI Projects SDK
- OpenAI Client
- Project Endpoint
- Azure CLI Authentication

## Step-by-Step Process
1. Deploy GPT model.
2. Configure endpoint and deployment name.
3. Authenticate with Azure CLI.
4. Create AIProjectClient.
5. Create OpenAI client.
6. Call Responses API.
7. Display output.

## Code Blocks
Includes examples using AIProjectClient, DefaultAzureCredential, and responses.create().

## Actionable Takeaways
- Use Azure CLI for local development.
- Store endpoints in environment variables.
