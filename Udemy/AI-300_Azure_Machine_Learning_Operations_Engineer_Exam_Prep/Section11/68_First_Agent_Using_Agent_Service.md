# First Agent Using Agent Service

## Summary
Walkthrough of creating an agent through the Foundry portal and exploring tracing, evaluations, tools, knowledge, and memory.

## Key Concepts
- Agent Versioning
- Conversation ID
- Tracing
- AI Quality
- File Search
- Memory Store

## Definitions
### Conversation ID
Identifier used to track and maintain conversation history.

### Memory Store
Mechanism used to persist context across sessions.

## Step-by-Step Process
1. Create agent.
2. Select model.
3. Enter instructions.
4. Save version.
5. Chat with agent.
6. Review traces.
7. Add tools.
8. Upload knowledge.

## Actionable Takeaways
- Use conversation IDs for continuity.
- Leverage tracing for debugging.
