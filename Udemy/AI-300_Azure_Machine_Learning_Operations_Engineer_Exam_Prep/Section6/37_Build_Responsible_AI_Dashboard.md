# Lab: Build a Responsible AI Dashboard

## Key Concepts
- Responsible AI Dashboard
- Error Analysis
- Causal Analysis
- Feature Importance
- Pipeline-Based Dashboard Construction

## Definitions
### Responsible AI Dashboard
A collection of explainability and evaluation tools used to analyze machine learning models.

### Treatment Feature
A feature selected for causal-effect analysis.

### Dashboard Constructor
A pipeline component used to generate Responsible AI insights.

## Examples
- Diabetes MLflow model.
- Diabetes MLTable training and testing data.
- Causal analysis on glucose, BMI, insulin, and blood pressure.

## Step-by-Step Process
1. Open the Azure ML model registry.
2. Select a registered MLflow model.
3. Open the Responsible AI tab.
4. Create a new dashboard.
5. Select training and testing data.
6. Specify the model type.
7. Enable error analysis.
8. Enable feature importance.
9. Configure causal-analysis features.
10. Configure dashboard settings.
11. Submit the dashboard pipeline.
12. Review and troubleshoot pipeline execution.
13. Resubmit if necessary.
14. Wait for dashboard generation.
