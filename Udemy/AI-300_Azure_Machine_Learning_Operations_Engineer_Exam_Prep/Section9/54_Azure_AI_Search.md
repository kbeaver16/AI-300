# Azure AI Search Introduction

## Summary
Azure AI Search provides indexing, enrichment, vector search, and retrieval capabilities for enterprise AI solutions.

## Key Concepts
- Data Source
- Skillset
- Indexer
- Index
- AI Enrichment
- Vector Search

## Definitions
### Skillset
Pipeline that performs text and image enrichment.
### Index
Searchable database containing processed content.

## Examples
- PDF processing
- Image analysis
- Keyword extraction
- Enterprise search

## Step-by-Step Process
1. Connect data source.
2. Configure skillset.
3. Run indexer.
4. Populate index.
5. Query index.

## Actionable Takeaways
- Use AI Search as the knowledge layer for RAG.
- Leverage enrichment for text and image content.
