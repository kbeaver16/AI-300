# Evaluation Metrics

## Summary
Overview of BLEU, ROUGE, Levenshtein, BERTScore, groundedness, relevance, coherence, and fluency metrics.

## Key Concepts
- Reference-based Metrics
- Reference-free Metrics
- LLM-as-Judge

## Definitions
### BLEU
Lexical overlap metric.
### ROUGE
Sequence overlap metric.
### BERTScore
Semantic similarity metric.
### Groundedness
Measures support from provided context.

## Examples
- RAG groundedness scoring.
- Semantic similarity comparisons.

## Actionable Takeaways
- Use semantic metrics alongside lexical metrics.
- Evaluate both grounding and fluency.
