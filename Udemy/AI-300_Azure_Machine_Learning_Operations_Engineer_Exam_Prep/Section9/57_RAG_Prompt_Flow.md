# Implementing a RAG Prompt Flow

## Summary
Build a Prompt Flow that uses Azure AI Search retrieval and GPT generation.

## Key Concepts
- Index Lookup
- Hybrid Search
- Semantic Search
- Generation Node

## Step-by-Step
1. Connect AI Search.
2. Create Prompt Flow.
3. Add Index Lookup node.
4. Configure search settings.
5. Add Generation node.
6. Inject retrieved context.
7. Return final answer.

## Examples
- Query travel offerings.
- Ask about hotels in San Francisco.

## Actionable Takeaways
- Use hybrid + semantic retrieval.
- Limit top-k documents appropriately.
