# AI Search Index Setup

## Summary
Create a multimodal Azure AI Search index using PDF text, images, GPT vision descriptions, and embedding models.

## Key Concepts
- Multimodal RAG
- Image Vectorization
- Embeddings
- Semantic Ranker
- Indexer

## Definitions
### Semantic Ranker
Improves document ordering based on semantic meaning.
### Image Vectorization
Generate text descriptions from images then embed them.

## Step-by-Step
1. Deploy embedding model.
2. Create AI Search index.
3. Connect Blob Storage.
4. Enable multimodal processing.
5. Connect Foundry models.
6. Configure image storage.
7. Enable semantic ranking.
8. Build index.

## Actionable Takeaways
- Use semantic ranking for better retrieval.
- Process images as well as text.
