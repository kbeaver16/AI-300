# AI Agents and Compound AI Systems

## Key Concepts
- AI Agents
- Compound AI Systems
- Tool Calling
- Planning
- External Systems
- APIs
- Runtime Decision Making

## Definitions
### Compound AI System
A system that combines an LLM with additional components such as APIs, tools, and external systems.

### AI Agent
An AI system that can reason about tasks, plan actions, and use tools dynamically.

### Tool
A callable capability such as an API, database query, retrieval function, or external action.

### Planning
The process of deciding which actions should be executed and in what order.

## Examples
- LLM interacting with APIs.
- Agents retrieving information from external systems.
- Business process automation workflows.

## Step-by-Step Processes
### Evolution Toward Agents
1. Build chatbot using an LLM.
2. Connect the LLM to external systems.
3. Add tool-calling capabilities.
4. Allow runtime planning.
5. Create autonomous agent behavior.

### Agent Execution Flow
1. Receive a user request.
2. Determine intent.
3. Select tools.
4. Plan execution sequence.
5. Call external systems.
6. Gather results.
7. Generate final response.

## Code Blocks
```text
Agent = LLM + Instructions + Tools + Planning
```
