# Introduction to Generative AI

## Key Concepts
- Artificial Intelligence (AI)
- Machine Learning (ML)
- Deep Learning
- Generative AI
- Large Language Models (LLMs)
- Foundation Models
- Transformer Architecture

## Definitions
### Artificial Intelligence
A field of computer science focused on creating systems that can replicate or exceed human intelligence.

### Machine Learning
A subset of AI that learns patterns from data to make predictions or decisions.

### Deep Learning
A subset of machine learning based on neural networks.

### Generative AI
AI systems that generate new text, images, audio, or other content from prompts.

### Large Language Model (LLM)
A neural-network-based model trained to understand and generate language.

### Foundation Model
A large pre-trained model such as GPT or Llama that serves as the base for downstream AI applications.

## Examples
- ChatGPT
- Microsoft Copilot
- Google Gemini
- GPT-4
- Llama models

## Step-by-Step Processes
### Evolution of AI
1. Artificial Intelligence (1956).
2. Machine Learning (1997).
3. Deep Learning (2012).
4. Generative AI (2021+).

### How an LLM Works
1. Receive user instructions.
2. Process input through transformer layers.
3. Analyze linguistic relationships.
4. Predict likely next tokens.
5. Generate a response.
