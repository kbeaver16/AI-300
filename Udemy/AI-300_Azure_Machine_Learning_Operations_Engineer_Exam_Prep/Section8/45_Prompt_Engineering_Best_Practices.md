# Prompt Engineering: Introduction and Best Practices

## Key Concepts
- Prompt Engineering
- Goal
- Context
- Expectations
- Source Material
- Prompt Specificity

## Definitions
### Prompt Engineering
The practice of designing clear and precise prompts to maximize the quality of AI outputs.

### Goal
The desired outcome of the prompt.

### Context
Background information needed to complete the task.

### Expectations
Instructions describing output style, tone, or format.

### Source
Reference information the model should use.

## Examples
### Poor Prompt
Tell me about history.

### Better Prompt
Explain the causes and effects of World War II in a clear and concise manner.

### Image Generation Prompt
Detailed instructions describing appearance, packaging, style, logos, and formatting.

## Step-by-Step Processes
### Building an Effective Prompt
1. Define the goal.
2. Provide context.
3. Specify expectations.
4. Supply source materials.
5. Review for clarity and specificity.

### Prompt Framework
- Goal: What do you want?
- Context: Why do you need it?
- Expectations: How should it respond?
- Source: What information should be used?

```text
Goal + Context + Expectations + Source
```
